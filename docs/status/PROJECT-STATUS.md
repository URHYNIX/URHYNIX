- 제외 범위: FR5 로봇팔, 픽앤플레이스, 실제 장애물 제거
- 포함 범위: SLAM/Nav2, LiDAR 장애물 대응, 카메라 영상 인식, Unity 관제 UI, DB 기록, 발표용 비교 데모
