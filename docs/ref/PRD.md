# PRD

## 한 줄 소개

Unity Digital Twin 관제 화면에서 TurtleBot3의 자율주행, 장애물 인식, 카메라 인식 결과를 확인하는 프로젝트입니다.

## 왜 만들까?

움직이는 로봇이 주변 장애물을 어떻게 인식하고 대응하는지 일반인도 화면으로 이해할 수 있게 만들기 위해서입니다.

## 누가 쓸까?

- 재난 대응 로봇을 연구하는 사람
- 자율주행/로봇 데이터를 분석하는 사람
- 로봇 시연을 보는 일반 관람자
- 카메라 인식과 LiDAR 기반 장애물 대응을 비교하고 싶은 사람

## 이번에 만들 것

- TurtleBot3 SLAM/Nav2 기반 주행
- LiDAR 기반 장애물 감지와 정지/회피
- 카메라 기반 장애물 인식
- Unity Digital Twin 관제 UI
- 주행 결과와 인식 결과 DB 저장
- 발표용 비교 데모: LiDAR only vs Camera + Vision

## 이번에는 안 만들 것

- FR5 로봇팔 제어
- 픽앤플레이스
- 장애물 물리 제거
- 완전 자동 재난 구조 시나리오

## 성공 기준

- TurtleBot3가 목적지까지 이동한다.
- 장애물을 감지하면 회피하거나 정지한다.
- 카메라 영상에서 장애물 종류와 신뢰도를 얻는다.
- Unity 화면에서 로봇 위치, 경로, 상태, 카메라 인식 결과가 보인다.
- DB에 주행 시간, 정지 횟수, 인식 결과가 저장된다.
- 발표 때 일반인이 "로봇이 무엇을 보고 어떻게 반응했는지" 이해할 수 있다.
