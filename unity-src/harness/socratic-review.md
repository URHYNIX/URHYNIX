# Socratic Review

## Questions

1. 이 변경의 목적은 무엇인가
2. 현재 verified scope를 어디까지로 잠가야 하는가
3. 가장 위험한 runtime/live failure는 무엇인가
4. 대안은 무엇인가
5. 되돌리기 어려운 결정은 무엇인가
6. 어떤 검증이 있어야 green이라고 말할 수 있는가

## Verify

- 최소 5개 질문 이상 답변
- 대안 1개 이상 검토
- scope expansion이면 관련 success pattern 또는 roadmap에 판단을 남김
