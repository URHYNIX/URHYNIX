# Session Retro

## Capture

- 이번 세션에서 잘 먹힌 절차
- 반복된 수동 체크
- 다음에 command/skill/harness로 승격할 만한 것
- 문서 또는 검증에서 아직 번거로운 부분

## Output

- `PROJECT-STATUS` 또는 active handoff에 1줄
- 필요 시 `harness/REGISTRY.md` 또는 `.claude/commands/` 후보
- 필요 시 자동화 후보 메모
