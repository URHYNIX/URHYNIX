# Visualization/RobotControl

RobotControl 씬 전용 시각화 드라이버.

## 포함 대상
- FR5/Fairino 전용 joint driver
- pose/path preview helper
- RobotControl 전용 overlay

## 새 구조 기준 하위 폴더
- `Drivers/` — joint driver, runtime rig binding
- `Preview/` — 3D preview, ghost/path preview
- `Overlays/` — target/limit/local annotation overlay
