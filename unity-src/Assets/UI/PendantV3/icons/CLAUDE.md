# Assets/UI/PendantV3/icons/

Pendant V3 아이콘 에셋 폴더.

## 규칙
1. 파일명은 `icon-{feature}.png` 패턴을 기본으로 쓴다.
2. 정적 아이콘은 `PendantV3.spriteatlas`에 묶는다.
3. 아이콘 의미가 바뀌면 README/feature 문서의 기능 매핑도 같이 맞춘다.

## 현재 매핑 기준
- `home` → NavRail Home
- `motion` → 조작
- `points` → 포인트
- `io` → I/O
- `status` → 상태
- `help` → 도움
