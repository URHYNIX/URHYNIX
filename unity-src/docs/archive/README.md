# docs/archive/

완료되었거나 현재 active 탐색 순서에서 내려놓은 문서를 보관한다.

## 하위 폴더

- `completed/`: 종료된 보드, handoff, 과거 상태판
- `legacy/`: 구조상 남아 있지만 현재 기준선으로 쓰지 않는 문서

## 규칙

- 현재 판단 SSOT는 두지 않는다.
- active 문서에서 archive로 보낼 때는 원 경로에서 새 기준 문서를 링크한다.
- history가 필요할 때만 참조한다.
