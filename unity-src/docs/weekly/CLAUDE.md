# docs/weekly/

주간 롤업 문서. `docs-nightly-organizer` 자동화가 자동 생성.

## 파일명 컨벤션
- `YYYY-WNN.md` (예: `2026-W10.md`)

## 규칙
- 수동 생성 가능하나, 자동화 실행 시 덮어쓰기됨.
- 주간 요약 형식: Phase 진행률, 완료 항목, 다음 주 목표.
