## 구현 노트
- 카메라 입력: Unity `WebCamTexture` 또는 외부 SDK
- 검출: 외부 비전 서비스 또는 Unity Barracuda/Sentis
- 좌표 변환: 카메라 캘리브레이션 → 로봇 좌표계
- P2 범위에서 인터페이스 정의만, 실제 비전 연동은 별도 프로젝트
