## AI 보조 유형

### 1. 실시간 경고 (Deterministic)
LLM 불필요 — 수학적 계산 기반.

```text
┌─ AI 경고 배너 ───────────────────────────────┐
│                                               │
│  ⚠ "현재 자세는 특이점(singularity)에        │
│     가깝습니다. J5 = 89.5°"                  │
│                                               │
│  💡 "J5를 85° 이하로 조정하면 특이점을       │
│     피할 수 있습니다."                        │
│                                               │
│  [무시]  [J5 조정하기]                        │
│                                               │
└───────────────────────────────────────────────┘
```

| 경고 유형 | 트리거 | 메시지 예 |
|-----------|--------|-----------|
| 특이점 근접 | J5 ≈ 0° 또는 ±180° | "특이점에 가깝습니다" |
| 관절 한계 | Joint > 90% 범위 | "J3이 한계에 근접합니다" |
| 큰 이동 거리 | 이동 > 200mm | "이동 거리가 큽니다 (342mm)" |
| 충돌 예상 | 경로 간섭 감지 | "이 경로에 충돌 위험이 있습니다" |
| 속도 과다 | 속도 > 50% + 근접 작업 | "현재 속도가 높습니다" |

### 2. 상황 추천 (Deterministic + Rule-based)
```text
┌─ ActionHint (ContextPanel) ──────────────────┐
│                                               │
│  💡 추천                                      │
│                                               │
│  "이 위치를 Pick 포인트로 저장할까요?"        │
│  (3회 연속 같은 위치로 이동함)                │
│                                               │
│  [포인트 저장]  [무시]                        │
│                                               │
└───────────────────────────────────────────────┘
```

| 추천 유형 | 트리거 | 제안 |
|-----------|--------|------|
| 포인트 저장 제안 | 같은 위치 3회 이동 | "포인트로 저장?" |
| 루프 변환 제안 | 반복 패턴 감지 | "반복 작업으로 변환?" |
| 속도 조정 제안 | 정밀 작업 구간 | "속도를 낮추는 게 좋겠습니다" |
| 좌표계 전환 제안 | Tool 방향 작업 | "Tool 좌표계가 더 편할 수 있습니다" |

## 자연어 명령 (P2 후속)

### 의도 기반 UI
```text
┌─ 빠른 명령 ──────────────────────────────────┐
│                                               │
│  [명령 입력: 현재 위치를 홈 아래 대기점으로 저장]│
│                                               │
│  해석 결과:                                    │
│  → 현재 위치를 "홈 아래 대기점"으로 저장      │
│                                               │
│  [실행]  [수정]  [취소]                       │
│                                               │
└───────────────────────────────────────────────┘
```

### 지원할 자연어 패턴 (P2)

| 패턴 | 변환 결과 |
|------|----------|
| "현재 위치를 XX로 저장" | 포인트 저장 (name=XX) |
| "XX와 YY 사이를 왕복" | 시퀀스 생성 (MoveJ→XX, MoveJ→YY, Loop) |
| "그리퍼 닫고 2초 대기 후 위로" | IO + Wait + MoveL(Z+) |
| "이 작업을 N회 반복" | 시퀀스 Loop 설정 |
| "속도 10%로 천천히" | Speed Preset 변경 |

## 구현 우선순위

| 단계 | 기능 | 기술 |
|------|------|------|
| P0 | 특이점/한계 경고 | Deterministic (기존 계산) |
| P1 | 포인트 저장/루프 추천 | Rule-based |
| P2 | LLM 설명 + 자연어 명령 | LLM API |

> LLM은 **설명과 추천만** 담당. 실제 로봇 명령의 결정권은 deterministic 계층에 둠.
