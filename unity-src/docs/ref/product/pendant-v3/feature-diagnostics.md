## 에러 메시지 3단 구조

기존 펜던트는 에러 코드만 보여주는 경우가 많다.
V3는 **3단 구조**로 초보자도 대응할 수 있게 한다.

```text
┌─ 3단 에러 메시지 ────────────────────────────┐
│                                               │
│  1단: 무슨 상태인지                           │
│  "충돌이 감지되었습니다"                      │
│                                               │
│  2단: 왜 그런지                               │
│  "로봇 링크가 작업공간 경계 또는 외부         │
│   물체와 접촉했습니다"                        │
│                                               │
│  3단: 지금 무엇을 하면 되는지                 │
│  "1. 장애물 확인 → 2. 수동 이동 →            │
│   3. 오류 초기화 → 4. 안전 위치로 이동"       │
│                                               │
└───────────────────────────────────────────────┘
```

### 에러 코드 → 한국어 메시지 매핑

| 코드 | 1단 (상태) | 2단 (원인) | 3단 (해결) |
|------|-----------|-----------|-----------|
| E-1023 | 충돌 감지 | 링크가 경계/물체와 접촉 | 장애물 확인 → 수동 빼기 → 초기화 |
| E-1001 | 통신 끊김 | 네트워크 연결 불안정 | 케이블 확인 → 재연결 |
| E-1015 | 관절 한계 초과 | J{n}이 허용 범위를 벗어남 | 반대 방향으로 이동 → 초기화 |
| E-1030 | 서보 과전류 | 부하가 허용치를 초과 | 부하 확인 → 재시작 |
| E-1050 | 비상정지 활성 | E-Stop 버튼이 눌림 | E-Stop 해제 → 초기화 |

> `FairinoErrorTranslator`에서 코드→메시지 변환. V3에서도 동일 로직 재사용.

## 복구 안내 카드 (ContextPanel)

### 컨텍스트 기반 동적 안내
```text
┌─ 다음 행동 추천 ──────────────┐
│                                │
│  💡 [동적 메시지]              │
│                                │
│  상태별 예시:                  │
│  - 미연결: "연결을 먼저 해주세요" │
│  - 서보OFF: "서보를 켜세요"    │
│  - 첫 접속: "Sync를 먼저       │
│    실행하세요"                 │
│  - Fault: "오류 초기화를       │
│    누르세요"                   │
│  - 정상: "조작 탭에서          │
│    로봇을 움직여보세요"        │
│                                │
│  [실행 →]                      │
│                                │
└────────────────────────────────┘
```

### 안내 우선순위

| 우선순위 | 조건 | 메시지 |
|----------|------|--------|
| 1 | Fault 발생 | "에러를 먼저 해결하세요" + [오류 초기화] |
| 2 | Safety Stop | "안전 정지 상태입니다" + [해결 순서 보기] |
| 3 | 미연결 | "로봇에 연결하세요" + [연결] |
| 4 | 서보 OFF | "서보를 켜세요" + [서보 켜기] |
| 5 | 미동기화 | "현재 위치를 동기화하세요" + [Sync] |
| 6 | 정상 대기 | "조작 탭에서 로봇을 움직여보세요" |

## 이벤트 로그 리스트

### ListView 아이템 구조
```text
┌─ 이벤트 아이템 ──────────────────────────┐
│  [아이콘]  [시간]  [메시지]              │
│                                          │
│  아이콘 종류:                            │
│  ✓ 성공 (AccentSuccess)                 │
│  ⚠ 경고 (AccentWarning)                │
│  ⛔ 에러 (AccentDanger)                 │
│  ● 정보 (AccentPrimary)                 │
│                                          │
│  시간 형식: HH:mm:ss (로컬 시간)        │
│                                          │
│  필터: [전체] [에러만] [경고만] [동작만] │
└──────────────────────────────────────────┘
```
