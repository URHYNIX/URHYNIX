## Pick & Place 템플릿 위저드

```text
┌─ Pick & Place 설정 ──────────────────────────┐
│                                               │
│  Step 1/4: Pick 위치 설정                     │
│                                               │
│  방법: (●) 현재 위치 사용  ( ) 좌표 입력      │
│                                               │
│  현재 위치:                                    │
│  X: -320  Y: 150  Z: 280                     │
│  [이 위치를 Pick으로 설정]                    │
│                                               │
│  Approach 높이: [50 ] mm (Pick 위 대기 높이)  │
│                                               │
│  [이전]                          [다음 →]    │
│                                               │
├───────────────────────────────────────────────┤
│  Step 2/4: Place 위치 설정                    │
│  Step 3/4: 그리퍼 설정                        │
│  Step 4/4: 속도/반복 설정                     │
└───────────────────────────────────────────────┘
```

### 생성되는 시퀀스
```
1. MoveJ → Approach (Pick 위 50mm)
2. MoveL → Pick
3. Wait 0.3s
4. IO: 그리퍼 닫기
5. Wait 0.5s
6. MoveL → Approach (Pick 위 50mm)
7. MoveJ → Approach (Place 위 50mm)
8. MoveL → Place
9. Wait 0.3s
10. IO: 그리퍼 열기
11. Wait 0.3s
12. MoveL → Approach (Place 위 50mm)
13. MoveJ → Home
```
