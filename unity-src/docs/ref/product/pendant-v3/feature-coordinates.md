## CoordStrip (ContextPanel 상단, 항상 표시)

### 와이어프레임

```text
┌─ CoordStrip ──────────────────┐
│                                │
│  관절 (deg)                    │
│  ┌──────┬──────┬──────┐       │
│  │ J1   │ J2   │ J3   │       │
│  │ 0.0  │-32.0 │ 84.0 │       │
│  ├──────┼──────┼──────┤       │
│  │ J4   │ J5   │ J6   │       │
│  │ 0.0  │ 90.0 │ 0.0  │       │
│  └──────┴──────┴──────┘       │
│                                │
│  TCP (mm, deg)   좌표계: Base  │
│  ┌──────┬──────┬──────┐       │
│  │ X    │ Y    │ Z    │       │
│  │-497.0│-130.0│477.0 │       │
│  ├──────┼──────┼──────┤       │
│  │ RX   │ RY   │ RZ   │       │
│  │180.0 │ 0.0  │ 90.0 │       │
│  └──────┴──────┴──────┘       │
│                                │
│  [Joint] [TCP] [Both]  표시   │
│                                │
└────────────────────────────────┘
```

### 값 변경 시 강조
- 값이 변경되면 해당 셀 **0.5초간 `AccentPrimary` 배경** 플래시
- 한계 근접 시 **`AccentWarning` 텍스트** 전환
- 한계 초과 시 **`AccentDanger` 텍스트** 전환

### Tablet에서의 CoordOverlay
```text
┌─ CoordOverlay (3D 뷰포트 좌측 하단, 반투명) ──┐
│  J: 0.0  -32.0  84.0  0.0  90.0  0.0          │
│  T: -497  -130  477  180.0  0.0  90.0          │
│                                        [확대▶] │
└────────────────────────────────────────────────┘
```
- 한 줄 요약 모드 (공간 절약)
- [확대] 탭 시 BottomSheet로 전체 CoordStrip 표시

## 좌표계 시각화 (3D 뷰포트)

### 표시 가능한 좌표계

| 좌표계 | 위치 | 색상 | 기본 표시 |
|--------|------|------|-----------|
| Base (World) | 로봇 베이스 | X=Red, Y=Green, Z=Blue | ON |
| Tool | TCP (공구 끝) | X=Magenta, Y=Cyan, Z=Yellow | ON |
| User/Wobj | 사용자 정의 원점 | X=Orange, Y=Lime, Z=Purple | OFF |
| Joint | 각 관절 축 | 관절 색상 | OFF |

### 3D 툴바 (뷰포트 상단)

```text
┌─ 3D 툴바 ──────────────────────────────────────┐
│  [Base축] [Tool축] [User축] [관절축]           │
│  [궤적ON/OFF] [고스트ON/OFF] [카메라 리셋]     │
└────────────────────────────────────────────────┘
```

### 좌표축 시각화 상세
```text
         Z (Blue)
         │
         │
         │
         └──────── Y (Green)
        /
       /
      X (Red)

  축 길이: 100mm (고정)
  축 끝: 화살표 머리 (cone)
  레이블: 축 이름 + 방향 (X+, Y-, 등)
  투명도: 선택된 좌표계 100%, 비선택 30%
```

### 초보자 시각화 기능
1. **"왜 이 방향으로 움직이지?"** 해결:
   - TCP 조그 시 이동 방향 축을 **굵고 밝게** 강조
   - 나머지 축은 반투명
   - 화살표 끝에 "+방향" 라벨

2. **좌표계 차이 비교**:
   - Base축과 Tool축을 동시에 켜면 **두 좌표계의 방향 차이가 시각적으로 보임**
   - 초보자가 "Tool 좌표계에서 X+가 Base에서는 어느 방향인지" 직관적 이해

3. **관절 축 표시**:
   - 각 관절의 회전축을 링(arc) + 화살표로 표시
   - 관절 조그 시 해당 관절 축만 강조

## 좌표 단위/정밀도 규칙

| 값 | 단위 | 소수점 | 예시 |
|----|------|--------|------|
| 관절 각도 | deg (°) | 1자리 | 84.0° |
| TCP 위치 | mm | 1자리 | -497.0 mm |
| TCP 회전 | deg (°) | 1자리 | 180.0° |
| 증분 이동 | mm 또는 ° | 1자리 | 5.0 mm |

### 표시 형식
- 양수: 부호 없음 (`84.0`)
- 음수: `-` 접두사 (`-32.0`)
- 영점: `0.0` (0 아님)
- 단위는 헤더에 1회 표시, 각 값에는 생략
