# Derived Course Content Policy

## Purpose
- 비공개 강의 PNG를 직접 넣지 않고 파생 개념만 남기는 정책을 정의한다.

## Parent Doc
- [PRD](../../PRD.md)

## When To Read
- 강의자료를 Lesson, Glossary, Instructor UI로 녹일 때

## Locked Decisions
- 원본 강의 PNG는 저장소, 빌드, Git에 넣지 않는다
- 원본 이미지를 직접 인용한 UI는 금지한다
- 파생 개념, 쉬운 설명, UI 카드 요구사항만 문서화한다

## Open Questions
- 외부 배포 전 저작권 검토 프로세스를 문서로 더 분리할지

## Downstream Sync
- `docs/ref/PRD.md`
- `docs/ref/product/content/concept-to-ui-map.md`

## Last Updated
- 2026-03-11 (KST)

## Allowed
- 개념명
- 쉬운 설명 문장
- lesson 연결 정보
- UI 카드/오버레이/툴팁 요구사항

## Forbidden
- 원본 PNG 파일 추가
- 원본 PNG 파일명/로컬 경로 기록
- 원본 슬라이드 직접 캡처나 재배포
