# ai-context/

## 목적
AI 협업을 위한 최소 문서 세트. 최신 권위 문서에 대한 빠른 참조 제공.

## 읽기 순서
1. `ai-context/START-HERE.md`
2. `ai-context/master-plan.md`
3. `ai-context/project-context.md`
4. `ai-context/coding-guideline.md`

## 로그/아카이브
- 세션 로그: `ai-context/logs/`
- 마일스톤 아카이브: `ai-context/archive/YYYY-MM-DD/`

## 규칙
- 중복 문서 금지.
- 완료된 상세 문서는 archive/로만 이동.
- 활성 문서가 충돌 시 우선 적용.
